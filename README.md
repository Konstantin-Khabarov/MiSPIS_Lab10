# Университетская информационная система

## Описание проекта
Модель и прототип информационной системы университета для научно-педагогического состава. Проект включает UML-диаграммы, сгенерированный Java-код и демонстрационную программу.

Университетская информационная система (УИС) — это централизованная платформа, предназначенная для управления данными научно-педагогического состава университета. Система обеспечивает преподавателям, заведующим кафедрами, сотрудникам отдела кадров и администраторам удобные инструменты для взаимодействия и выполнения рабочих процессов.
Каждый преподаватель имеет персональную учётную запись в системе. После входа он может просматривать свою личную информацию (учёное звание, степень, ставка, кафедра), редактировать контактные данные, просматривать свою учебную нагрузку, а также подавать отчёты о научной деятельности. Преподаватель может отправлять заявки на отпуск, которые затем рассматривает заведующий кафедрой.
Заведующий кафедрой отвечает за распределение учебной нагрузки между преподавателями. Он может просматривать данные преподавателей своей кафедры, утверждать или корректировать учебную нагрузку, просматривать научные отчёты и принимать решение по заявкам на отпуск (одобрить или отклонить).
Сотрудник отдела кадров (HR) управляет кадровыми данными преподавателей. Он может регистрировать новых преподавателей, обновлять сведения о штатных единицах, переводах, наградах и повышениях квалификации. При необходимости HR может деактивировать учётную запись преподавателя в случае увольнения.
Администратор системы обеспечивает техническое сопровождение УИС: управляет правами доступа пользователей, конфигурирует роли, контролирует корректность данных, отвечает за резервное копирование и восстановление системы. Администратор также помогает HR и заведующим кафедрами в случае технических неполадок.
Система автоматически ведёт журнал событий, фиксируя все действия пользователей для обеспечения прозрачности и безопасности. Преподаватели и заведующие кафедрами могут получать уведомления о статусах заявок и отчётов.
